폰트

개요

  • 폰트는 5가지로 분류(명조, 고딕, 헤드라인, 손글씨, 일정간격)
  • MS, Google, Apple, 네이버에서 배포한 폰트가 주류
  • Google, 네이버는 무료 배포

폰트의 계보

폰트의 종류
계열명 국문 설명
serif 명조 아름답게 보일 목적으로 획의 끝 부분에 장식을 더함
sans-serif 고딕 sans=없다(프랑스). 세리프가 없는 글자
display 헤드라인 굵고 강렬한 문자
handwriting 손글씨 자유로운 글자, 타이포그라피
monospace 동일간격 모든 문자의 간격을 일정하게 유지(한글: 2글자 취급), ㅇㅇ체

* serif, sans-serif, monospace는 폰트명이 아니라, 폰트 계열명을 나타냅니다.
운영체제나 브라우저는 기본(default)문자열을 설정할 수 있습니다. 따라서, 여러개의 보조 폰트를 설정할 때는 기본 폰트가 가장 마지막에 등장해야 합니다.

serif

 명조(sarif)의 획 끝 부분은 휘어져 있지만, 고딕(sans-sarif)는 그렇지 않습니다.

monospace

 monospace는 모든 글자 및 띄어쓰기의 가로넓이가 동일하지만, 일반 글씨는 그렇지 않습니다. 영문의 경우, A와 i의 세로 간격이 다릅니다. 한글의 경우 'ㅇㅇ체'라고 불리는 폰트는 해당 폰트의 가로넓이를 일정하게 유지하는 성격으로 작성된 경우가 많습니다.
 monospace는 글자와 띄어쓰기의 크기가 일정하기 때문에, 특히 코딩영역에서는 아주 중요하게 받아들여집니다. 최근엔 영문과 국문이 혼용이 자연스러운 D2Coding으로 통일되는 것 같습니다.

폰트의 종류

국내

  • 명조 : 나눔명조, 본명조, 바탕, 굴림, 돋움
  • 고딕 : 맑은고딕, 나눔고딕
  • 헤드라인 : 휴먼둥근헤드라인
  • 손글씨 : 궁서, 휴면편지체, 나눔손글씨 펜 ...
  • 일정 간격 : D2Coding, 바탕체, 궁서체, 굴림체

국내에서는 Naver에서 발간한 나눔체로 보편화되었 있습니다. 무료이며, 종류도 다양(배포중인 것만 109종)합니다. 메이커는 '산돌(Sandoll)'로 나오고 있습니다.

국외

  • serif : Times new Roman, Garamond, Georgia
  • sans-serif : Arial, Helvetica, Verdana, Tahoma, Segoe UI, AppleSDGothicNeoR
  • Display : Impact, Segoe UI Black
  • handwrite : Brush Script MT, Monotype Corsiva, Lucida Handwriting ...
  • monospace : Consolas

수학공식

  수학공식의 서체는 'Times new Roman' 또는 Symbola 의 이텔릭체를 이용합니다. lateX에서 MJXZERO, MJXTEX와 같은 서체를 사용하기도 합니다.

설정 예시

  • 구글 : Roboto, 'Segoe UI',Arial,'Malgun Gothic',Gulim,sans-serif
    구글(유튜브): Roboto, Arial, sans-serif
  • 네이버 : -apple-system, BlinkMacSystemFont, "Malgun Gothic", "맑은 고딕", helvetica, "Apple SD Gothic Neo", sans-serif
    블로그: Helvetica-Light,"나눔고딕",nanumgothic,AppleSDGothicNeo,"malgun gothic","맑은 고딕",sans-serif
  • 다음 : AppleSDGothicNeo-Regular, Malgun Gothic, \b9d1\c740 \ace0\b515, dotum, \b3cb\c6c0, sans-serif
    티스토리 : se-nanumgothic,\\B098\B214\ACE0\B515,nanumgothic,sans-serif,Meiryo
  • MS(bing) : "Segoe UI",Segoe,Tahoma,Arial,Verdana,sans-serif

대부분 '애플폰트' → '네이버/구글 폰트' → 'MS폰트' → '시스템폰트' 순으로 배치된 것을 볼 수 있습니다. 아무래도 가장 보편적인 MS폰트를 가장 앞에 배치하면, 모두 MS폰트로 걸러지기 때문에, 최초 의도한 폰트이면서 덜 유명한 폰트부터 배치하는 것이 일반적입니다.
