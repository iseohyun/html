헤더Header : 상단 문서정보

  • 헤더 감추기: 스크롤 내림(또는 헤더의 "▲UP"버튼)
  • 헤더 보이기: 스크롤 올림
  • 문서 이동: 하나의 챕터에 여러개의 문서가 있을 수 있습니다.
    • 이전 문서 << 타이틀: 이전 문서가 없으면 나타나지 않습니다.
    • 타이틀 >> 이후 문서
    • 타이틀 : 보더에 site-map을 띄웁니다.
  • 아이콘icon
    •   : home화면으로 이동
    •   : 목차 보이기/감추기
    • "▲UP" : 헤더 감추기
    • show tutorial : (튜토리얼 코드가 있는 경우에만) 코드명 보이기/감추기

보더Board : 좌측 네비게이터

  • 목차 보이기/감추기 :
    • 열기: 보더가 안보이는 상태에서, 아무 곳이나 double click
    • 닫기: 보더가 열린 상태에서, 아무 곳이나 click
  • site-map ↔ 목차 전환 : '문서 제목↻' 클릭

빠른 복사copy to clipboard

  1. 코드블럭code, 인용블럭blockquote위에 마우스를 올립니다.
  2. 우상단에 "copy" 버튼이 나타납니다.
  3. 버튼을 클릭하면 클립보드에 복사되고, "클립보드에 복사되었습니다."는 메시지가 페이지 우하단에 표시됩니다.
  4. 복사하려는 곳에서 'ctrl + v'를 통해 붙여넣기 할 수 있습니다.
