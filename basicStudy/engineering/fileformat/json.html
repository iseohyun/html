Json(.json) 예시 [예시 링크:hierarchy.json] 

[
  {
  "기초학문" : [
    "디렉토리" : "project",
    "단원" : [
      { 
        "주제": "집합",
        "파일명":"set.html"
      },
      {
        "주제": "다항식",
        "파일명": "polynomial.html"
      }
    ]
  }
]
접근 방법
var requestURL = 'hierarchy.json';  // 저장된 파일을 불러온다. 
var request = new XMLHttpRequest();
request.open('GET', requestURL);
request.responseType = 'json';
request.send();                    // 파일 요청을 하고, 
request.onload = function () {     // 로딩이 완료되었을 때 동작을 정의 
  var file = request.response;
  var dir = file[0].단원[1].파일명   // dir에 "polynomial.html"을 저장
규칙
  • 집합set은 {}기호를, 배열array은 []을 사용합니다.
  • 집합은 key, value로 구성합니다.(map구조)
  • 원소간의 구분은 콤마(,)로 합니다.
  • 권고 : 하위 구조는 들여쓰기로 가독성을 지원합니다.(space:2)
