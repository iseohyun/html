# 베지어 곡선 시뮬레이터 (Bezier Curve Simulator)

본 프로젝트는 조절 가능한 제어점(Control Points)의 수와 위치에 따라 동적으로 수학적 베지어 곡선이 생성되고 수렴하는 선형 보간의 애니메이션 궤적 과정을 시각화하여 보여주는 교육용 웹 시뮬레이터 프로그램입니다.

---

## 📂 디렉터리 구성 및 모듈 설명

```
small-project/Bezier/
├── index.html            # UI 마크업, 에러 콘솔 탑재 및 외부 모듈 연결
├── Bezier.css            # 스타일 격리 가이드 준수 CSS 시트
├── version.md            # 변경 기록 및 버전 히스토리
├── readme.md             # 본 가이드 문서
└── module/
    ├── config.js         # 전역 상태(state) 및 DOM 선택자 상수 관리
    ├── simulation.js     # 순수 선형 보간(lerp) 및 N차 베지어 포인트 수학적 계산 알고리즘
    ├── chart.js          # 제어점 동적 드로잉, 드래그 핸들링 및 SVG 골격선/궤적 패스 렌더러
    └── main.js           # 이벤트 바인딩, 브라우저 크기 대응, 애니메이션 타이머 제어
```

---

## 🚀 주요 기능 및 핵심 구조
1. **차수별 동적 렌더링 지원**:
   - 2개 제어점(1차 선형), 3개 제어점(2차 베지어 Q 커맨드), 4개 제어점(3차 베지어 C 커맨드) 및 5개 이상 제어점의 고차 베지어 궤적을 100등분 분할 근사선으로 일반화하여 매끄럽게 렌더링합니다.
2. **새로고침 없는 동적 리렌더링 (SPA)**:
   - 제어점 가감 및 리셋 버튼 클릭 시 기존의 `location.href` 호출을 통한 강제 깜빡임 리로드 문제를 제거하고 데이터 배열의 추가/삭제를 통해 제어점 SVG 노드를 실시간 동적 재구축합니다.
3. **주소창 실시간 좌표 동기화**:
   - 브라우저를 수동으로 새로고침(F5) 하더라도 현재 사용자가 이동시킨 점들의 위치를 온전히 복구해내기 위해 `history.replaceState` API를 활용하여 화면 깜빡임 없이 주소창의 `?p=...` 좌표 매개변수를 실시간 동기화합니다.
