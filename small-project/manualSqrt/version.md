### [v1.0.7] - 설명 고도화
- 참조1: 연한 핑크
- 참조2: 연한 파랑
- target: 노랑(기존)
'333'의 제곱긍 예시
기존: "2. x의 제곱 중, 3을 넘지 않는 가장 큰 x는 12 입니다."
변경: "x^2 < 3을 만족하는 x = 1 입니다."
추가설명: 위 설명에서 3은 참조1 이다. 3의 source에 해당하는 칸에 하이라이트(참조1), 말풍선의 위치는 target의 아랫쪽입니다.

기존: "3. x = 1를 작성합니다."
수정: "1를 작성합니다."
추가설명: 말풍선은 아랫쪽을 향해야 합니다.

기존: "4. 1 × 1를 구합니다."
수정: "1 x 1 = 3"
추가 설명: 처음등장하는 1= 참조1, 두번째 등장하는 1=참조2, 마지막 3= target
target은 1자리수이므로 1자리만 하이라이트 되어야 합니다. 이미 3이 입력되어 있어야 합니다.
참조 1(왼쪽)과 참조2(위, 몫 중 1에 해당하는 부분)은 하이라이트 되어 있어야 합니다.

기존: "5. 333에서 100을 뺍니다. (3 - 100_raw = 2, 2을 입력함)"
수정: "3 - 1 = 2"
추가 설명: 3은 참조1, 1은 참조2, 2는 타겟, 2는 이미 작성되어있어야 하고, 1자리이므로 하이라이트도 1개만 적용

기존: "6. 뒤에 '33'을 붙입니다."
수정: "'33'을 붙입니다."
추가 설명: 33은 참조1(피제수에서 해당 부분 2자리수만 하이라이트)

기존 : "7. 1 + 1 = 1"
수정 : "1 + 1 = 2"
추가설명: ... "2는 이미 작성되어 있어야 한다."

기존 : 8. (퀴즈내용 유지)
수정: (퀴즈내용 유지)
추가설명: 이미 작성한 2의 위치는 유지한체로..

기존: "9. 제수 아래에 x = 8를 작성합니다."
수정: "8를 작성합니다."
추가 설명: 이미 8이 작성되어 있다.

기존: "10. 가장 뒷 자리에 8을 붙여 줍니다."
수정: "뒷 자리에 8을 붙여 줍니다."
추가: 이때 8은 참조1이다.

기존: "4. 28 × 8를 구합니다."
수정: "28 × 8 = 224"
추가 설명: 28은 참조1, 8은 참조2

### [v1.0.6] - UI개선
1. 처음 입력 숫자를 넣고 엔터를 치면 focus out
2. 단축키 모달(Modal)은 팝업 형태로 출력하고 x를 클릭하여 닫기
3. 단축키 표
  |기능|단축키|보조단축키|
  |다음단계|Enter|->|
  |이전단계|shift + Enter| <-|
  ...
4. 소수점이 1칸을 차지하므로 다음 연산과정에서도 같은 column은 공백으로 유지하여 줄 맞춤

### [v1.0.4] - 키 바인딩
- keybinding.js를 생성하고 단축키를 이용한 제어가 가능합니다.
  - Enter, 방향키(->)를 이용해 다음으로 넘어갈 수 있습니다.
  - Enter는 primary, ->는 secondary로 지정합니다. 이 키는 사용자가 변경할 수 있습니다.
  - Shift-Enter, <-를 통해 이전으로 갈 수 있습니다.
  - Space를 통해 skip할 수 있습니다. skip이 없을 때, 다음으로 넘어가는 기능을 수행합니다.
  - Esc를 누를 때 초기화합니다.

- 자릿수가 내려오거나 연산이 추가될 때의 연출을 부드러운 애니메이션 효과로 개선합니다.
