### [v1.0.4] - 키 바인딩
- keybinding.js를 생성하고 단축키를 이용한 제어가 가능합니다.
  - Enter, 방향키(->)를 이용해 다음으로 넘어갈 수 있습니다.
  - Enter는 primary, ->는 secondary로 지정합니다. 이 키는 사용자가 변경할 수 있습니다.
  - Shift-Enter, <-를 통해 이전으로 갈 수 있습니다.
  - Space를 통해 skip할 수 있습니다. skip이 없을 때, 다음으로 넘어가는 기능을 수행합니다.
  - Esc를 누를 때 초기화합니다.

### [v1.0.3] - 단계 이동 편의 및 아키텍처 확장
- [이전] (Prev) 버튼을 추가하여 단계를 자유롭게 전후로 이동할 수 있도록 합니다.
- 자릿수가 내려오거나 연산이 추가될 때의 연출을 부드러운 애니메이션 효과로 개선합니다.
- 고정된 연산 줄 개수 제한(max_line = 8)을 해제하여, 사용자가 원하는 자릿수까지 무한히 내려가며 계산할 수 있도록 구조를 확장합니다.

### [v1.0.2] - 인터페이스 개선
- "Next" 버튼을 따로 두지 않고(삭제), 매번 툴팁에서 다음 버튼(">")을 통해서 진행
- "hide" 버튼과 기능을 삭제
- "Language" 버튼과 기능을 삭제

### [v1.0.1] - 디자인 개선
- 현재 단계에 해당하는 항목을 강조
- 항목에 대한 설명이 항목 바로 밑에 생성
- 나눗셈의 각 단위들이 서로 세로줄이 맞도록 구성
  - 현재 하나의 input에 소숫점 n자리가 한번에 출력되고 있으나, 하나의 input에 하나의 자릿수만 출력하게 함으로써 정확하게 종횡이 맞도록 정렬
