### [v1.1.1] - 스타일 클래스 고유 식별자 격리(#janggi-app) 및 CSS 오염 차단
- **메인 래퍼 태그 고유 ID 격리 (#janggi-app)**: 기존에 bare 태그명 선택자 `article`에 부여되었던 전역 레이아웃 속성(`width: 100vw; height: 100vh; overflow: hidden;` 등)이 상위 블로그/SPA 사이트의 공통 `article` 요소로 스타일이 누출되어(Bleeding), 대국 화면 로딩 시 본문 본체의 스크롤을 무력화하고 댓글("이 문서에 대한 질문 및 건의") 및 일부 네비게이션 영역만 화면에 잠긴 채 갇히게 만들던 심각한 CSS 오염 오류를 수정. 메인 장기판 전체를 `<article id="janggi-app">` 고유 선택자로 격리 및 수정함으로써 상위 SPA 플랫폼 내 완벽한 스타일 격리(Sandboxing) 보장

### [v1.1.0] - 실시간 모니터링 우측 사이드바 설정창 이식 및 1버튼 진영 양방향 교대(toggleNation) 도입
- **실시간 모드 우측 사이드바 설정창 구축 (Split-view)**: 데스크톱 해상도(width >= 850px) 기준, 설정창이 온 스크린을 덮어 가리던 방식에서 우측 450px 크기로 슬라이딩되는 사이드바 레이아웃으로 변경. 설정창 활성화 시 왼쪽의 장기판 뷰포트 영역이 이에 연동되어 실시간으로 축소/재배치되며, 설정창 조작(기물 크기 조정, 색상 변경, 모양 변환 등)에 따른 변화를 왼쪽 보드를 통해 100% 실시간으로 모니터링할 수 있도록 획기적 개선 (모바일 화면에서는 기존의 풀스크린 오버레이 모드로 자동 스위칭)
- **1클릭 진영 양방향 스위칭 (toggleNation)**: 초/한 나라 중 어느 쪽 진영 버튼을 클릭하더라도 아군/적군 진영이 깔끔하게 즉시 양방향으로 회전교대(toggle)하도록 핸들러를 단일화하고, 클릭 시 미니어처 예시 배치도까지 양방향으로 동시 회전 및 배색 반전되도록 최적화

### [v1.0.7] - 3D 빗면 베벨 입체감 조율, 윗면 원색 테두리 제거, 베지어 곡선 기반 3D 들었다 놓기 애니메이션 및 들기 고도 제어
- **3D 외곽 베벨 및 윗면 원색 테두리 삭제**: 3D 입체 팔각형 기물의 윗면 페이스를 좌상단 방향으로 3px 편향 배치해 우하단 빗면(Bevel Side) 입체 그림자 영역을 강화하고, 페이스 테두리를 따라 그려져 시야를 분산시키던 한나라(빨강), 초나라(파랑)의 얇은 원색 안쪽 테두리 팔각형 라인을 기물 32개 전체에서 전면 삭제해 극도의 미니멀리즘과 붓글씨의 조화를 도모
- **마우스 호버 모션 정적화**: 마우스 커서를 장기알에 올렸을 때 우상향으로 튀어나와 뷰포트를 뒤흔들던 번잡한 호버 트랜스폼 모션을 비활성화하고, 차분한 drop-shadow 증폭 하이라이트 효과만 남겨 대국 몰입도 증진
- **베지어 곡선(Bezier Curve) 기반 3D 들었다 놓기(Lift-and-Drop) 애니메이션**: 기물 이동 및 기보 탐색 시 순간이동되던 연출을 개선하여, GPU 하드웨어 가속 CSS transform(translate)과 `requestAnimationFrame`을 연동해 이동 거리에 비례하는 둥근 궤적(2차 베지어 곡선 아치)을 그리는 비행 애니메이션 구현
- **동적 고도 스케일 및 다이나믹 그림자**: 이동 중간 지점(50% 고도)에서 기물 크기가 최대 1.18배로 커진 뒤 안착 시 1배로 축소되고, 비행 높이에 따라 바닥의 그림자 블러와 오프셋 거리가 실시간으로 팽창 및 축소 투사되는 정밀 3D 입체 렌더링 구현
- **기물 렌더링 순서 동적 정렬 (z-index 보장)**: 기물이 공중으로 비행할 때 다른 정적 기물이나 표시선 밑으로 들어가는 렌더링 순서 역전을 해결하기 위해, `setPieces` 호출 시 이동 중인 기물 노드를 SVG의 마지막 자식으로 DOM 트리에 동적 재배치(`appendChild`)하여 항상 최고 레이어로 표현되도록 개선
- **설정창 내 애니메이션 속도 및 들기 높이 슬라이더 탑재**: 설정판 내에 애니메이션 이동 시간(0.0~2.0초, 기본 0.5초) 조절 바와 함께 기물 들기 높이 배율(0.0~2.0, 기본 0.2) 조절 바를 추가 탑재하여 기물 비행 아크의 높이와 팽창 수준을 실시간 커스텀 가능하게 통합

### [v1.0.3] - 반응형 뷰박스 스케일링, 뷰포트 물리 높이 캡핑, 비동기 레이스컨디션 해결 및 선택자 매칭 버그 수정
- **정밀 타겟 선택자 매칭 (`#janggi-svg`)**: SPA 숙주 모드에서 좌측 네비게이션 바의 홈 아이콘 SVG를 장기판으로 잘못 가져와 레이아웃 조작이 오동작하던 버그를 해결하기 위해 장기판에 고유 ID를 부여하고 선택식을 개정
- **뷰포트 물리 잔여 높이 감지**: 부모 컨테이너가 고정 높이 없이 유동적으로 늘어날 경우 스크린 세로 영역을 벗어나 하단이 잘리던 현상을 방지하기 위해 뷰포트 내 남은 물리적 크기(`window.innerHeight - topOffset`)를 계산식의 한계값으로 적용

- **비동기 스크립트 실행 순서 보장**: SPA 라우터 dynamic script insertion 시 `async = false`를 부여해 순차 실행을 보장하고, 메인 엔트리에서 리소스 로드 대기열 폴링 (`checkAndInit`)을 탑재해 `pieces is not defined` 등의 레이스 컨디션 예외를 완벽 해결
