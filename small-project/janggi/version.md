### [v1.0.0] - 역할별 모듈 분할 및 구조화 (SoC)
- 기존 단일 `janggi.html` 구조에서 HTML, CSS, JavaScript 관심사 분리(SoC)를 위한 모듈 구조화 진행
- 기본 진입점을 `index.html`로 변경하고 스타일 코드를 `style.css`로 완전히 이관
- JavaScript 코드를 역할에 따라 `config.js`, `logic.js`, `render.js`, `main.js`로 분리 및 로드 순서 최적화 (`defer` 사용)

### [v1.0.1] - 3D 입체 8각 기물 및 오리지널 서예풍 글씨 개편
- **오리지널 서체 누끼 적용**: 초(초서체) 및 한(해서체) 고유 서체를 보존하기 위해 원본 PNG 이미지 배경을 투명하게 정밀 크로마키(누끼) 가공하여 적용
- **3D 입체 8각 토큰**: 양측 기물 모두 입체 베벨 효과가 반영된 3D 팔각 토큰으로 통합 디자인 (한나라 기물에는 나이테 음각 디테일 추가)
- **그림자 스타일 정밀 연동**: 투사 그림자 오프셋과 흐림 강도를 조정하여 예시 이미지의 초나라 장기알 깊이감을 동일하게 구현
- **네온 발광 효과**: 기물 선택 시 황금색 아우라 발광, 이동 경로 후보지 표시 시 네온 블루 링 효과 적용
- **반응형 모션**: 마우스 호버 시 기물이 3D 입체로 살짝 튀어나오는 마이크로 인터랙션 모션 추가

### [v1.0.3] - 반응형 뷰박스 스케일링, 뷰포트 물리 높이 캡핑, 비동기 레이스컨디션 해결 및 선택자 매칭 버그 수정
- **정밀 타겟 선택자 매칭 (`#janggi-svg`)**: SPA 숙주 모드에서 좌측 네비게이션 바의 홈 아이콘 SVG를 장기판으로 잘못 가져와 레이아웃 조작이 오동작하던 버그를 해결하기 위해 장기판에 고유 ID를 부여하고 선택식을 개정
- **뷰포트 물리 잔여 높이 감지**: 부모 컨테이너가 고정 높이 없이 유동적으로 늘어날 경우 스크린 세로 영역을 벗어나 하단이 잘리던 현상을 방지하기 위해 뷰포트 내 남은 물리적 크기(`window.innerHeight - topOffset`)를 계산식의 한계값으로 적용
- **동적 헤더 마진 클리어링**: 상단 고정 헤더 영역의 유무 및 높이를 감지해 `#control-box`에 동적으로 `margin-top` 여백을 주어 헤더 뒤로 게임판이 숨겨지는 버그 방지
- **비동기 스크립트 실행 순서 보장**: SPA 라우터 dynamic script insertion 시 `async = false`를 부여해 순차 실행을 보장하고, 메인 엔트리에서 리소스 로드 대기열 폴링 (`checkAndInit`)을 탑재해 `pieces is not defined` 등의 레이스 컨디션 예외를 완벽 해결
