초고속 무손실 영상 분할기

Github : https://github.com/iseohyun/mp4-trim

고화질 영상도 빠르게 분할하는 파이썬 Standalone 프로그램입니다. 최근 영상작업을 하는데, 편집 프로그램에 올린 후에 작업하려니 너무 무겁기도 하고, 영상 코덱이 변하는 문제가 발생하기도 해서 만들어보았습니다.

  • - 업로드 된 내용은 ffmpeg.exe를 포함하고 있지 않습니다.
  • - Readme.md에 ffmpeg.exe를 다운받는 방법이 있으니 참고하시기 바랍니다.
  • - 파이썬은 공식 홈페이지에서 다운받으시고, 동봉된 build.bat를 실행하시면 exe파일로 변환됩니다.
