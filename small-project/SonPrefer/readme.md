# 남아선호사상 시뮬레이션 (Son Prefer Simulation)

남아가 태어날 때까지 계속 출산하는 관습(남아선호사상)이 실제로 사회 전체의 출생 성비를 바꾸는지 검증해보는 교육용 웹 시뮬레이터입니다.

## 💡 시뮬레이션 개요
수학적으로 매 출생은 독립 시행(남:여 확률은 항상 보존됨)이므로, 각 가정이 남아가 나올 때까지 계속해서 아이를 낳더라도 사회 전체의 최종 남녀 성비는 자연 성비와 완전히 동일하게 나타납니다. 

이 프로젝트는 대규모 난수 생성을 통해 이 현상을 직접 실험하고, 시각적 차트로 이를 증명하여 직관적인 이해를 돕습니다.

- **남아선호사상 조건**: 남아를 낳을 때까지 계속 출산합니다 (가구당 평균 약 1.95명).
- **1가구 2자녀 조건 (비교군)**: 가구당 성별에 상관없이 무조건 2명의 자녀를 둡니다 (가구당 정확히 2명).

## 🛠️ 기술 및 파일 구조
`AI_Coding_Rules.md`에 명시된 **표준 SPC 모듈화 아키텍처**를 기반으로 작성되었습니다.

- **`index.html`**: UI 컨트롤러 폼, 결과 노출 영역 및 SVG 차트 컴포넌트를 정의하고 에러 콘솔을 상시 장착합니다. 가구수와 성비를 입력할 수 있는 로그 스케일 슬라이더(Range)와 숫자 필드가 탑재되어 있으며, 하단 "2. 시뮬레이션 원리 및 설명" 영역에 관련 수학적/생물학적 논증이 통합되어 있습니다.
- **`SonPrefer.css`**: SPA 호스트 환경과 격리하기 위해 `body`, `article` 레이아웃을 상속(Inherit)받도록 설정하고, 독립 클래스(`.simulation-container`) 내에서만 스타일을 한정합니다.
- **`module/config.js`**: 공통 상수, 색상 테마 및 런타임 데이터 구조를 정의합니다.
- **`module/simulation.js`**: DOM 요소 조작을 일절 포함하지 않고 오직 독립 시행 난수 연산을 통해 데이터 결과를 생성합니다. (남아 출생 시 루프를 올바르게 종료하도록 버그가 수정되었습니다.)
- **`module/chart.js`**: 계산된 데이터 값을 기반으로 SVG 막대를 그리고, 10ms 단위의 매끄러운 높이 충전 애니메이션 및 그리드라인 스케일링을 관리합니다.
- **`module/main.js`**: 애플리케이션 진입점 및 입력 바인딩(숫자 창 - 슬라이더 간의 양방향 로그 스케일 연동), 이벤트 조율을 관장합니다.

## 🚀 사용 방법
1. 브라우저에서 `index.html` 파일을 엽니다.
2. **가구수(테스트 횟수)**와 **성비**를 슬라이더 또는 숫자 입력 필드를 사용하여 조절합니다. 
   - 가구수 범위: 100가구 ~ 100억가구 (로그 스케일 적용)
   - 성비 범위: 80 ~ 120 (기본값: 105)
3. **[테스트 실행]** 버튼을 클릭합니다.
4. 애니메이션과 함께 시뮬레이션이 실행되고 결과 패널 및 그래프가 자동으로 갱신됩니다.
