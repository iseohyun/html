### [v1.4.0] - 2026-07-22
- **Phase 1 극형식 시각화 고도화 완수**: 복소수 곱셈이 직각좌표계보다 압도적인 기하학적 우위를 갖는 핵심 원리인 **각도의 덧셈($\theta_3 = \theta_1 + \theta_2$)**과 **크기의 곱셈($r_3 = r_1 \times r_2$)**을 증명하는 보조 시각 레이어를 탑재했습니다.
  - **각도 덧셈 꼬리물기 호 (Arc Chaining)**: 파란색 Z1 호 끝에서 초록색 Z2 각도만큼 이어서 그려지는 점선 연결 호(`chain-arc-path`)를 드로잉하여 회전 각도의 기하학적 덧셈 성질을 직관적으로 보여줍니다.
  - **닮음비 삼각형 (Similarity Triangles)**: 기준 삼각형 $\triangle O E Z_1$ ($E=(1,0)$)과 결과 삼각형 $\triangle O Z_2 Z_3$ 의 반투명 다각형 레이어를 대조 렌더링하여 $1 : r_1 = r_2 : r_3$ 의 SAS 기하학적 닮음 관계 및 크기 곱셈의 원리를 시각적으로 증명합니다.

### [v1.3.0] - 2026-07-22
- **원점 대칭 벡터 기반 툴팁 자동 정렬**: 제어점 드래그 시 표시되는 실시간 좌표 툴팁의 배치 메커니즘을 전면 고도화했습니다. 툴팁이 포인트 자체나 텍스트 라벨을 가리지 않도록 **원점(중앙)으로부터 그 점을 향해 뻗어 나가는 방향 벡터**를 실시간으로 연산하여, 제어점의 먼 바깥 방향으로 툴팁을 밀착 배치합니다. 제어점을 360도 회전시킬 때 툴팁도 궤도를 따라 제어점 바깥쪽을 부드럽게 감싸며 자동 공전 정렬됩니다.

### [v1.2.0] - 2026-07-22
- **실시간 좌표 툴팁(Tooltip) 추가**: 제어점 Z1, Z2를 드래그 조작할 때 해당 점의 수학적 복소수 좌표($x+yi$), 크기(Magnitude), 각도(Angle, 0~360도 형식) 정보를 보여주는 실시간 툴팁 팝업을 탑재했습니다. 이 툴팁은 마우스 업(드래그 종료) 후에도 화면에 **계속 노출**되며, 툴팁 우상단의 `×` 버튼을 클릭하여 개별적으로 닫을 수 있습니다. 줌(wheel) 및 브라우저 크기 조절 시에도 툴팁 위치가 제어점 우상단에 밀착하여 자동 추종하도록 리포지셔닝 로직을 장착했습니다.
- **불필요한 reload 제거**: 실시간 URL 동기화로 인해 불필요해진 `reload` 버튼 및 `handleReload()` 함수를 삭제하고, 페이지 진입(새로고침) 시 $t$ 매개변수 및 슬라이더의 초기값을 `0`으로 세팅되도록 변경했습니다.

### [v1.1.0] - 2026-07-22
- **마우스 휠(wheel) 줌인/줌아웃 탑재**: 상단의 번거로운 Zoom range 슬라이더를 삭제하고, SVG 캔버스 안에서 마우스 휠을 굴릴 때 실시간으로 줌인/줌아웃(30 ~ 300 범위 제한)되도록 사용자 인터페이스를 전면 고도화했습니다. { passive: false } 옵션을 활용해 브라우저 기본 본문 스크롤 현상을 완벽히 방어했습니다.

### [v1.0.0] - 2026-07-22
- **표준 SPC Modular Architecture 개편**: 기존 단일 파일 `imaginary.html` 구조를 index.html, imaginary.css, 그리고 기능별 module(config, simulation, chart, main) 구조로 완전 이관 분리했습니다.
- **새로고침 깜빡임 버그 완벽 수정**: Reload 및 Zoom, 드래그 시 실제 페이지가 리로드되지 않고, replaceState와 해시 보존 URL 동기화 기법을 적용하여 무깜빡임 실시간 URL 갱신을 완성했습니다.
- **눈금자(Ruler) 및 드래그 십자 가이드라인(Guideline) 탑재**: 상단/좌측 좌표 눈금자 및 포인트 드래그 시 실시간 투사되는 십자 가이드라인 보조선을 탑재했습니다. 눈금자와 겹치지 않도록 제어점 최소 경계 제약(X>=35, Y>=25)을 적용했습니다.
- **SPA 렉시컬 격리 및 리스너/옵저버 누수 파괴**: 전역 블록 변수를 var 선언자로 완전 대체하고, 페이지 이탈 시점에 윈도우 리스너가 안전하게 해제되도록 청소 연동했습니다.
