# 복소평면 시뮬레이터 (Imaginary Simulator)

이 프로젝트는 복소수 곱셈 $Z_3 = Z_1 \times Z_2$ 의 기하학적 연산 과정을 복소평면 위에서 시각화하는 인터랙티브 웹 도구입니다.

## 주요 기능
1. **복소수 곱셈의 기하학적 의미 시각화**: 두 복소수 제어점($Z_1, Z_2$)의 곱셈 연산 결과($Z_3$)의 크기와 각도 변화를 동적으로 추적 및 렌더링합니다.
2. **Phase 1 극형식 기하 레이어 (각도 덧셈 호 & 닮음 삼각형)**:
   - **각도 덧셈 꼬리물기 호**: 파란 호($Z_1$) 끝에 초록 호($Z_2$)가 이어붙어 $Z_3$ 각도를 이루는 기하학적 각도 덧셈 성질($\theta_3 = \theta_1 + \theta_2$)을 보여줍니다.
   - **닮음비 삼각형**: 기준 삼각형 $\triangle O E Z_1$ ($E=(1,0)$)과 결과 삼각형 $\triangle O Z_2 Z_3$ 이 완벽히 닮음($1 : r_1 = r_2 : r_3$)임을 시각화하여 크기 곱셈 원리를 증명합니다.
3. **실시간 컨트롤러 및 애니메이션**: 재생/일시정지 단추를 통해 $t$ 매개변수 값의 변화에 따른 회전 궤적 애니메이션을 감상할 수 있습니다. (초기값 $t=0.0$)
4. **마우스 휠(wheel) 줌인/줌아웃**: 그리드 내부에서 마우스 휠 스크롤을 조작하여 직관적으로 복소평면 배율을 확대/축소할 수 있습니다.
5. **실시간 제어점 정보 툴팁**: 제어점 드래그 시 좌표($x+yi$), 크기(Magnitude), 각도(Angle)가 툴팁으로 실시간 노출됩니다. 툴팁은 **원점으로부터 제어점을 잇는 바깥 방향으로 자동 회전 배치**되어 좌표 점과 글씨를 가리지 않으며, 우상단 `×`를 눌러 개별 닫을 수 있습니다.
6. **좌표 눈금자(Ruler) 및 십자 가이드라인(Guideline)**: 상단/좌측 정밀 픽셀 눈금자와 포인트 드래그 시 드래그 선을 관통하는 실시간 가이드라인을 제공합니다.
7. **무깜빡임 실시간 URL 위치 보존**: 새로고침 없이 상태 변경(Zoom, Drag, Reload) 시 브라우저의 쿼리를 실시간 업데이트하고 해시를 보존하여 연동합니다.

## 표준 디렉토리 구성
- `index.html`: 메인 뷰 진입점 및 에러 콘솔 탑재.
- `imaginary.css`: 스타일 시트 격리 관리.
- `module/config.js`: 전역 변수 및 상수 관리.
- `module/simulation.js`: 순수 복소수 연산 공식 및 보간 알고리즘 (DOM 격리).
- `module/chart.js`: SVG 캔버스 구성, 보조선, 가이드라인, 눈금자, 닮음 삼각형 및 실시간 툴팁 생성.
- `module/main.js`: 수명 주기 관리 및 이벤트 연결.
