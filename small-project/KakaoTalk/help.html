카카오톡 대화 저장 방법

  1. 대화 중인 카톡방에 들어간다.
  2. 오른쪽 상단의 "메뉴(삼선표시)"를 연다.
  3. 오른쪽 상단의 "설정(톱니바퀴)"를 누른다.
  4. "대화내용 내보내기"를 누른다.
  5. "텍스트 메시지만 저장"을 누른다.
  6. 현재 사이트에서 "대화내용 불러오기"를 누른다.

세팅방법

  • 2인일 경우 대화상대를 소거하여 나를 특정합니다.
  • 단톡방의 경우 나를 알 수 없습니다. "환경설정"에서 "내 이름 (나)"를 설정합니다.
  • 대화가 길어질 경우 스크롤이 가능합니다. 다만, "애니메이션 설정"에서 "스크롤 고정"을 해제해 주세요.
  • 음성 설정은 "환경설정"의 "대화상대 초상화"에서 설정합니다. (아직은 시스템 기본 TTS만 지원해요.)
