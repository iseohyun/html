개요

최근 유튜브는 광고 차단 프로그램(Ad Blocker)에 대한 강력한 방어 기작을 도입하여, 일반적인 광고 차단 확장 프로그램을 사용할 경우 동영상 재생이 제한되거나 경고 팝업이 노출됩니다.

이 프로젝트는 유튜브의 광고 요소 자체를 차단하여 리디렉션을 막거나 차단 필터에 걸려 감지되는 전통적인 방식 대신, 유튜브 비디오 플레이어 내부 상태를 실시간 감지하여 사람이 직접 건너뛰기 버튼을 누르는 것처럼 이벤트를 시뮬레이션하고, 건너뛰기가 불가능한 광고는 강제로 배속을 극대화(16배속)하여 음소거 처리하는 방식으로 우회 동작을 수행하는 가볍고 강력한 크롬 확장 프로그램입니다.

작동 원리

유튜브 광고 스킵의 핵심 작동 메커니즘은 다음과 같은 3가지 단계로 이루어져 있습니다.

1. MutationObserver를 활용한 실시간 DOM 감시

전통적인 인터벌 폴링(setInterval) 방식은 매 밀리초 단위로 요소를 검색하여 무거운 연산을 반복하므로 CPU 자원을 많이 소모합니다. 본 프로그램은 브라우저의 MutationObserver API를 사용하여 유튜브 비디오 플레이어(.html5-video-player) 내부의 DOM 구조 변경사항만 추적해, 광고 상태 클래스(.ad-showing)가 감지되는 즉시 정교하게 작동합니다.

2. 마우스 이벤트 시뮬레이션 (isTrusted 우회)

유튜브는 단순 element.click()과 같은 자바스크립트 명령이 사람이 누른 것인지 확인하기 위해 event.isTrusted 속성을 확인하여 차단할 수 있습니다. 이를 우회하기 위해 실제 마우스 동작과 동일한 mousedown → mouseup → click 이벤트를 차례대로 생성 및 전달(Dispatch)하여 브라우저의 자동화 감지 기작을 완벽하게 우회합니다.

3. 16배속 고속 우회 및 음소거 (Unskippable Ad Bypass)

5초 대기 후에도 건너뛰기 버튼이 제공되지 않는 강제 광고나 일시적으로 제어할 수 없는 상황에서는, 비디오 태그의 playbackRate를 최대 속도인 16.0배속으로 올리고 muted = true로 음소거하여, 광고가 시작하자마자 0.5초 이내에 강제로 종료 단계(currentTime = duration - 0.1)로 이동시킵니다.

핵심 코드

실제 확장 프로그램의 백그라운드와 페이지 레벨에서 작동하는 핵심 스크립트 소스 코드입니다.

content.js (광고 감지 및 스킵 로직)

// 이벤트 위임 및 마우스 클릭 시뮬레이션 (isTrusted 방어 우회)
function simulateClick(element) {
    const eventOpts = { bubbles: true, cancelable: true, view: window };
    element.dispatchEvent(new MouseEvent('mousedown', eventOpts));
    element.dispatchEvent(new MouseEvent('mouseup', eventOpts));
    element.dispatchEvent(new MouseEvent('click', eventOpts));
}

// 광고 감지 및 스킵 메인 로직
function checkAndSkipAds() {
    if (!enabled) return;

    const player = document.querySelector('.html5-video-player');
    const video = document.querySelector('video');

    if (player && player.classList.contains('ad-showing')) {
        adPlaying = true;

        // 1단계: 플레이어 내부 건너뛰기 버튼 즉시 클릭
        const selectors = [
            '.ytp-ad-skip-button-modern',
            '.ytp-skip-ad-button',
            '.ytp-ad-skip-button',
            '[class*="ytp-ad-skip-button"]',
            '[class*="ytp-skip-ad-button"]',
            'button[aria-label*="건너뛰기"]'
        ];

        for (const selector of selectors) {
            try {
                const btn = player.querySelector(selector);
                if (btn && (btn.offsetWidth > 0 || btn.offsetHeight > 0)) {
                    simulateClick(btn);
                    break;
                }
            } catch (e) {}
        }

        // 2단계: 백업 우회책 (16배속 가속 및 음소거)
        try {
            if (video) {
                if (video.playbackRate < 16) {
                    video.playbackRate = 16;
                    video.muted = true;
                }
                if (isFinite(video.duration) && video.currentTime < video.duration - 0.1) {
                    video.currentTime = video.duration - 0.1;
                }
            }
        } catch (e) {}
    } else {
        // 광고 상태 해제 시 카운터 업데이트
        if (adPlaying) {
            adPlaying = false;
            chrome.storage.local.get(['skippedCount'], (res) => {
                const count = (res.skippedCount || 0) + 1;
                chrome.storage.local.set({ skippedCount: count });
            });
        }
    }
}

// DOM 감시 옵저버
const observer = new MutationObserver(() => { checkAndSkipAds(); });
observer.observe(document.documentElement, { childList: true, subtree: true });

popup.js (대시보드 컨트롤러)

document.addEventListener('DOMContentLoaded', () => {
  const toggleSwitch = document.getElementById('toggle-switch');
  const skipCounter = document.getElementById('skip-counter');
  const resetButton = document.getElementById('reset-button');

  // 상태 로드
  chrome.storage.local.get(['enabled', 'skippedCount'], (result) => {
    toggleSwitch.checked = result.enabled !== false;
    skipCounter.textContent = result.skippedCount || 0;
  });

  // 스위치 변경
  toggleSwitch.addEventListener('change', (e) => {
    chrome.storage.local.set({ enabled: e.target.checked });
  });

  // 통계 초기화
  resetButton.addEventListener('click', () => {
    if (confirm('스킵 누적 통계를 초기화하시겠습니까?')) {
      chrome.storage.local.set({ skippedCount: 0 }, () => {
        skipCounter.textContent = 0;
      });
    }
  });
});

설치 및 사용 방법

이 확장 프로그램은 크롬 웹 스토어를 통하지 않고 로컬 환경에서 다음과 같이 수동으로 개발자 모드로 즉시 로드하여 실행할 수 있습니다.

  1. 저장소 복제(Clone) 혹은 소스코드를 다운로드합니다. 
    git clone https://github.com/iseohyun/youtube-auto-skip-ad.git
  2. 구글 크롬(Chrome) 브라우저를 열고 주소창에 chrome://extensions를 입력해 이동합니다.
  3. 우측 상단의 개발자 모드 (Developer mode) 토글을 켭니다.
  4. 좌측 상단에 노출되는 압축해제된 확장 프로그램을 로드합니다. (Load unpacked) 버튼을 클릭합니다.
  5. 다운로드하여 압축을 해제한 프로젝트의 폴더(manifest.json 파일이 위치한 폴더)를 선택하여 등록합니다.
  6. 이제 유튜브를 실행하고 동영상을 재생하면 실시간으로 광고가 자동으로 스킵되는 대시보드 팝업창을 확인하실 수 있습니다.
