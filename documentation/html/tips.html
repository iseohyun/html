짜증나는 .(Dot) 에밋

문장을 작성하고 마침표. + 엔터enter를 누르는 순간 <div>테그가 열립니다. 재빨리 실행취소(ctrl + z)를 누르고 다시 ., 방향키를 누릅니다. 참 귀찮습니다. 그리고 정확히 5분후에 똑같은 짓을 반복합니다.

기본 에밋은 수정할 수도 없고, 그렇다고 에밋을 기능을 끄자니 아쉽습니다. 그냥 사용하자니, 화가납니다. 이 때는 . 에밋을 또 정의 하면 됩니다.

에밋 커스터마이즈는 이 문서를 참조하십시오. 

"dot":{
"prefix": ".",
"body": [
"."
],
"description":"dot is dot"
}
